Create a surreal, interactive, meme-driven website for a Tunisian streamer named M3kky and his community called "OGs". The website must be in English but reflect Tunisian humor, irony, and chaotic meme culture.

GENERAL STYLE:
The tone is playful, absurd, and troll-like, but the design should still feel smooth, modern, and interactive. The experience should feel like a mini-game or interactive story, not a traditional website.

IMAGES:

* m3kky.png: A close-up image of a young male streamer’s face with expressive, meme-worthy features. The nose and mouth area must be clearly visible because it will be used for a zoom-in transition.
* nattoun.png: A front-facing image of a dog with a distinct, expressive face. This represents "President Nattoun", a humorous fictional leader. The dog may appear serious in a funny way.

IMPORTANT:
Use m3kky.png ONLY as the homepage entry visual and zoom animation.
Use nattoun.png ONLY as a character inside the world (dialogues, popups, speeches).
Never swap their roles.

HOMEPAGE EXPERIENCE:

* Display m3kky.png full screen.
* No traditional menu.
* When the user clicks anywhere, trigger a smooth zoom into the nose.
* Transition the zoom into a surreal world inside the mouth.

WORLD CONCEPT:
Inside the mouth, reveal a fictional place called "Bahamas Land".
This world should look like a cartoonish map or small town with clickable locations (like a game map).

INTRO SCENE:

* A character (using nattoun.png) appears in a popup.
* He introduces himself as:
  “I am President Nattoun, leader of Bahamas Land.”
* Tone: serious but obviously ridiculous.
* Add funny lines like:
  “This is a serious country. Please behave.”
  “We are watching you.”

USERNAME SYSTEM:

* On first visit, ask the user to enter a username.
* Store it locally (localStorage/session).
* Use the username dynamically across the site:

  * Greetings from Nattoun
  * Messages like: “Welcome, [username]. We don’t trust you yet.”

MAP LOCATIONS (CLICKABLE AREAS):

1. COURT OF OGs:

* Users can submit short text (roasts or judgments about usernames).
* Display messages as “official verdicts”.
* Add random system verdicts like:

  * “GUILTY of being MID”
  * “INNOCENT but suspicious”
  * “BANNED (not really)”
* Add cooldown and character limits.

2. MUSEUM OF OGs:

* Users can upload images (memes) or short captions.
* Display in a chaotic gallery layout.
* Add funny labels:

  * “Ancient Artifact”
  * “Certified Mid”
* Optional: like/upvote system renamed to “Respect”.

3. LIBRARY (TROLL VERSION):

* Shows social media links (Twitch, TikTok, etc.)
* Labels should be misleading for humor:

  * “Forbidden Knowledge” → Twitch
  * “Do Not Click” → Instagram

4. BANK:

* Fake currency called “Nattoun Coins”
* Fake investment buttons:

  * “Invest Now”
  * On click → joke message like “You lost everything.”

5. PRESIDENT PALACE:

* Show nattoun.png giving speeches
* Random rotating messages:

  * “Everything is under control.”
  * “Nothing is under control.”

6. SECRET AREA:

* Hidden clickable zone
* Leads to unexpected or absurd content
* Easter eggs or inside jokes

INTERACTIONS:

* Add hover effects on map locations
* Add random popups occasionally:

  * “[username], stop clicking everything.”
* Custom cursor (optional: dog paw or meme icon)
* Smooth animations between sections

AUDIO (OPTIONAL):

* Meme sounds on clicks
* Short voice lines for Nattoun

MODERATION RULES:

* Limit text length (e.g., 100–150 characters)
* Basic content filtering
* Limit image size uploads
* Optional report button

PROGRESSION:
The experience should feel layered:

* First click → surprise zoom
* Then exploration → discover locations
* Then deeper → hidden jokes and secrets

GOAL:
Create a unique, funny, community-driven interactive website that feels like an inside joke for OGs, while still being understandable for new visitors.
